Positioning of elements in CSS

Positioning the elements of an HTML page is no easy feat. CSS provides a set of methods that can be used cumulatively:

  • Margin and spacing via the margin and padding properties
  • Centering: vertical or horizontal via the text-align a vertical-align
    properties
  • Display Type: display property
  • Positioning: position property
  • Floating Object: float property

1. Margin and Spacing

Margins and Spacing are used as a priority to position a block with respect to another block or with respect to its parent block.
The margin property is used to apply outer spacing to the block and the padding property applies inner spacing to the block.

A paragraph of text with an inner spacing of 5 pixels all around the text (gray dotted border) wrapped in a parent block (blue border) and vertical outer spacing of 15 pixels and horizontal outer spacing of 10 pixels. 

                        
                          <div style="border: 2px solid blue;">
                            <p style="padding: 5px; margin: 15px 10px; border: 1px dashed gray;">
                              A paragraph of text with an inner spacing of 5 pixels all around the text (gray dotted border) wrapped in a
                              parent block (blue border) and vertical outer spacing of 15 pixels and horizontal outer spacing of 10 pixels. 
                            </p>
                          </div>

2. Horizontal and Vertical centering

2.1 Horizontal centering

To center a text horizontally, the text-align property will be used. To center a block horizontally, one method is to use the automatic margins technique (margin: auto-cf.above. Another is to make use of flexible boxes with the justify-content property.)

A paragraph of text with an width of 50% that of the parent block,
centered horizontally, and whose text is also centered horizontally.

                            
                              <div style="border: 2px solid blue;">
                                <p style="padding: 5px; border: 1px dashed gray; text-align: center; width: 50%; margin: auto;">
                                A paragraph of text with an width of 50% that of the parent block,
                                centered horizontally, and whose text is also centered horizontally. 
                                </p>
                              </div>

2.2 Vertical centering

Vertical centering will depend on the container element.
If the container block has the display: table-cell property, such as a table for example, you can use vertical-align: middle Otherwise you can use
the flexible boxes and the align-items property.

Text paragraph 50% wide, centered
vertically and horizontally in its
parent block.

                            
                <div style="style="border: 2px solid blue; height: 250px; width: 500px; display: flex; align-items: center; justify-content: center;">
                    <p style="border: 1px dashed gray; text-align: center; width: 50%;">
                        Text paragraph 50% wide, centered vertically and horizontally in its parent block. 
                    </p>
                </div>

3. Display

Changing the rendering mode of HTML elements via the display property enables easy positioning of the elements relative to one another.
Display Block / Inline - Inline-block - Table cell - Flex (spacebetween).

3.1 Display Block/ Inline

By default, the main HTML elements use the block (div, p ... ) or inline (a, span ... ) renderings. It is possible to modify the rendering mode of an
element to change its display mode.
Example: use a bullet list containing links to generate a horizontal menu

  • Home
  • Section 1
  • Section 2
  • Section 3
  • Contact
                        
                            <nav>
                                <ul class="menu">
                                    <li><a href="#">Home</a></li>
                                    <li><a href="#">Section 1</a></li>
                                    <li><a href="#">Section 2</a></li>
                                    <li><a href="#">Section 3</a></li>
                                    <li><a href="#">Contact</a></li>
                                </ul>
                            </nav>

3.2 Display Inline-Block/Table-cell...

Other types of rendering modes allow, for example, inline and block display to be mixed: inline-block which allows a width, a height, margins and spacing to be added to an inline block.
Table-cell mode allows table properties to be applied to a block, generally used to center vertically.

                        
                            <div>
                                <label style="display: inline-block; width: 280px; color: green;">The inline-block label in my field</label>
                                <input type="email" name="email" placeholder="Your email"/>
                            </div>

3.3 Display Flex

It would take several laboratory exercises to cover all the possibilities presented by flexible boxes. This modern and very powerful rendering mode makes it possible to easily position elements relatives to each other.

                        
                            <div style="display: flex; justify-content: space-between; border: 2px solid blue; height: 80px;">
                                <div style="width: 30%; margin: 1.5%; border: 1px solid green;"></div>
                                <div style="width: 30%; margin: 1.5%; border: 1px solid pink;"></div>
                                <div style="width: 30%; margin: 1.5%; border: 1px solid gray;"></div>
                            </div>

4. Positioning Type

By default, all elements in the HTML flow are positioned statically=>position:static
It is possible to position elements by changing the value of the position property

4.1 Fixed Position

Fixed positioning allows an element to be taken out of the HTML flow and placed in a specific location by adjusting the top, bottom, right and left properties. This is the case, for example, of the header of this page, which has following CSS style:

                        
                            header.header-fix{
                                height: 85px;
                                width: 100%;
                                position: fixed;
                                top: 0;
                                left: 0;
                                background-color: #eee;
                                border-bottom: 1px solid #ddd;
                                box-shadow: 0px 3px 5px 0px rgba(0,0,0,0.75);
                            }

4.2 Relative Position

Relative posistioning keeps the element in the flow but allows it to be offset from its initial position via the left, right, top, and bottom properties.

This block is offset by 10 pixels down and
to the right
                        
        <div style="display: flex; justify-content: space-between; border: 2px solid blue; height: 80px;">
            <div style="width: 30%; margin: 1.5%; border: 1px solid green;"></div>
            <div style="width: 30%; margin: 1.5%; border: 1px solid pink; position: relative; top: 10px; left: 10px;">This block is offset by 10 pixels 
            down and to the right</div>
            <div style="width: 30%; margin: 1.5%; border: 1px solid gray;"></div>
        </div>

4.3 Absolute Position

Absolute positioning takes the HTML element out of the flow and positions it relative to the wrapping block (having the relative positioning property) according to the values provided by top, left, rigth and/or bottom.

This block is offset by 10 pixels down and
to the right with respect to the parent block
                        
    <div style="display: flex; justify-content: space-between; border: 2px solid blue; height: 80px; position: relative;">
        <div style="width: 30%; margin: 1.5%; border: 1px solid green;"></div>
        <div style="width: 30%; margin: 1.5%; border: 1px solid pink; position: relative; top: 10px; right: 330px;">This block is offset by 10 pixels
        down and to the right with respect to the parent block</div>
        <div style="width: 30%; margin: 1.5%; border: 1px solid gray;"></div>
    </div>
